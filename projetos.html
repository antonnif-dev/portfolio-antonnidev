Lex Link:

O Lex Link é uma plataforma web que aproxima empresas que precisam realizar diligências jurídicas de parceiros especializados prontos para executá-las, tudo de forma simples, organizada e supervisionada por um administrador.

O sistema foi desenvolvido para eliminar falhas de comunicação e desorganização no fluxo de tarefas jurídicas. Ele centraliza todo o processo em um só ambiente: as empresas cadastram diligências, os parceiros demonstram interesse, e o administrador define quem executará cada tarefa, com notificações automáticas garantindo que todos acompanhem o andamento em tempo real.

Continuo aprimorando o Lex Link para deixá-lo cada vez mais intuitivo, eficiente e agradável de usar, tornando a conexão entre empresas e parceiros uma experiência fluida e confiável.

O Blocos de Carnaval BH é um aplicativo leve e divertido que ajuda foliões a encontrarem os melhores blocos de rua durante o carnaval de Belo Horizonte.

O sistema foi desenvolvido com React Native, integrando dados locais e APIs para exibir informações atualizadas sobre blocos, horários, locais e trajetos. O foco está na experiência mobile fluida, com navegação simples, pesquisa rápida e mapas interativos, garantindo que o usuário encontre o bloco certo no momento certo.

O projeto segue em expansão, buscando oferecer cada vez mais funcionalidades e tornar a vivência do carnaval de BH mais conectada e organizada.

Acerto de contas:

é um aplicativo criado para ajudar pessoas a organizarem suas finanças pessoais de forma prática e visual, permitindo o controle de gastos e cobranças em um único lugar.

O app foi desenvolvido em React Native com armazenamento local via AsyncStorage, permitindo ao usuário criar tabelas personalizadas com descrições, valores, prazos e anexos. Cada item pode ser marcado como quitado ou removido, e o sistema organiza automaticamente os dados em histórico e ativos, com foco em segurança e performance mesmo offline.

Continuo otimizando o app com novas funcionalidades, transições suaves e uma experiência mobile-first cada vez mais intuitiva.

Lex Link:

O Quiz Frontend é um jogo interativo voltado para quem quer testar e aprimorar seus conhecimentos em HTML, CSS e JavaScript de forma divertida.

O projeto foi desenvolvido utilizando manipulação de DOM, arrays de perguntas dinâmicas e feedback instantâneo para cada resposta. A aplicação utiliza lógica condicional, eventos e temporizadores para simular uma experiência gamificada completa, com pontuação e resultados personalizados ao final.

É um projeto em constante evolução, pensado para unir aprendizado e entretenimento, tornando o estudo de frontend mais envolvente e prático.

Gerador de Currículo:

O Gerador de Currículo Online foi criado para facilitar a vida de quem precisa montar um currículo profissional rapidamente, sem depender de editores complexos.

O sistema utiliza formulários dinâmicos com JavaScript para coletar informações do usuário e gerar automaticamente um currículo formatado, pronto para download ou impressão. A aplicação trabalha com armazenamento local (localStorage) e manipulação de DOM para gerar o documento de forma ágil e responsiva.

O projeto continua recebendo melhorias para oferecer mais modelos, recursos de personalização e uma experiência ainda mais simples para quem busca destacar-se no mercado de trabalho.

ABC do Professor:

O ABC do Professor é um projeto educacional que auxilia docentes na criação de atividades, planejamentos e recursos interativos voltados ao ensino infantil e fundamental.
